# position_app

## Requirimientos  
  
  -PHP 8.1.0  
  -mysql  

## Inslatacion  

  - En la base de datos debe crear un una base de datos position_app  
      user:position_app  
      password:position  
      
  - Luego migrar la base de datos con el comando:  
      php artisan migrate  
      
  - Ejecutar servidor  
      php artisan serve
